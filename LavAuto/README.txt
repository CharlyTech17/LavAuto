Comment utiliser LavAuto : 
1. Assurez-vous que Java est installé (https://www.java.com/fr/download/) 
2. Double-cliquez sur LavAuto.bat pour lancer l'application 
3. Si l'application ne se lance pas, vérifiez que Java est bien installé 
4. En cas d'erreur avec la base de données, vérifiez la configuration dans le fichier config.properties
